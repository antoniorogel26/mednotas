# MedNotas — Apuntes Médicos con IA (versión Python)

## Requisitos
- Python 3.8+
- API key gratuita de [Groq](https://console.groq.com)

## Instalación

```bash
pip install -r requirements.txt
```

Edita el archivo `.env` y pon tu API key de Groq:
```
GROQ_API_KEY=gsk_TU_KEY_AQUI
```

## Ejecutar

```bash
python app.py
```

Abre el navegador en **http://localhost:5000**

## Notas
- Usa **Google Chrome** o **Microsoft Edge** para la grabación de voz
- La API key de Groq es gratuita en [console.groq.com](https://console.groq.com)
